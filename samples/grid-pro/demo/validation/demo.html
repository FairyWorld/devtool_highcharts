This demo showcases editable cells with custom validation rules. Doubleclick cells to edit the content and to see the validation in action.

To learn more read our documentation article on predefined and custom validation rules. String Validation
• Product Name: Accepts only letters, numbers, and spaces — no special characters.
• SKU: Must follow the format AAA-123 (3 uppercase letters, a hyphen, and 3 digits).

Number Validation
• Price: Must be zero or higher.
• Discount: Must be a number from 0 to 100.

Cross-Field Validation
• Discount: If Category is "Electronics", the discount must not exceed 30%.
