The demo shows how to export a Dashboards configuration to JSON. For demo purposes it's saved to local storage, but could e.g. be saved to a database and retrieved at a later stage.

Click the "Export dashboard" button below the dashboard or in the top right dropdown menu, and the JSON configuration object will be saved to local storage.

Click the "Destroy dashboard" button to destroy the current dashboard.

Click the "Import dashboard" button to create a new dashboard from the configuration saved in local storage.
